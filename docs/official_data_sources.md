# 国家能源局真实公开数据来源说明

本项目已内置一份基于国家能源局官网月度公告整理的真实公开数据：

- 文件：`data/official/national_electricity_consumption_monthly_nea.csv`
- 指标：全国全社会用电量（月度）
- 单位：亿千瓦时
- 时间范围：2024-03 至 2025-12

## 整理原则

- 优先使用国家能源局官网对应月份公告中的“当月全社会用电量”
- 对于未单独给出但可由官方累计值推导的月份，采用“年累计或 1-2 月累计减去已知当月值”的方式计算
- 所有推导月份在 `note` 字段中已明确标注

## 推导月份说明

- `2024-12`：由 `2024年全年全社会用电量 98521` 减去 `2024年1-11月累计 89686` 推导得到 `8835`
- `2025-01`：由 `2025年1-2月累计 15564` 减去 `2025年2月 7434` 推导得到 `8130`
- `2025-12`：由 `2025年全年全社会用电量 103682` 减去 `2025年1-11月累计 94602` 推导得到 `9080`

## 使用建议

- 课程演示时，可直接使用内置官方数据
- 论文或正式报告中，建议将 `source_url` 和整理方法一并列入附录
- 如果国家能源局后续发布了更新月份，可按同样 CSV 结构追加新行
