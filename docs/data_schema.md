# CSV 数据结构说明

## 必填字段

### `date`
- 含义：统计月份
- 类型：日期字符串
- 推荐格式：`YYYY-MM` 或 `YYYY-MM-DD`
- 示例：`2025-01`

### `consumption_billion_kwh`
- 含义：当月全社会用电量
- 类型：数值
- 单位：亿千瓦时
- 示例：`7300`

## 可选字段

### `source`
- 含义：数据来源
- 类型：字符串
- 示例：`国家能源局公开统计`

### `source_url`
- 含义：原始公告链接
- 类型：字符串
- 示例：`https://www.nea.gov.cn/...`

### `note`
- 含义：备注信息
- 类型：字符串
- 示例：`根据月度公告整理`

## 推荐示例

```csv
date,consumption_billion_kwh,source,note
2024-01,6900,国家能源局公开统计,根据公开月报整理
2024-02,6200,国家能源局公开统计,根据公开月报整理
2024-03,6700,国家能源局公开统计,根据公开月报整理
```

## 处理规则

- 系统会自动将日期归一化为每月第一天
- 系统会自动去除数值中的逗号和空格
- 如果存在重复月份，系统会按月份聚合并取均值
- 如果月份中间有缺口，系统会按时间顺序插值补齐
- 如果必填字段缺失，页面会报错并提示字段名

## 后续扩展建议

- 如果你后续要做省份或行业维度扩展，可以增加：
  - `region`
  - `sector`
  - `unit`
- 首版 MVP 不使用这些扩展字段
